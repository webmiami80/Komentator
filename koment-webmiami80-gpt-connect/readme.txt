=== Koment WebMiami80 GPT Connect ===
Autor: Paweł Zajączkowski
Wersja: 1.0.0
Email: webmiami80@gmail.com

== Opis ==

Wtyczka Koment WebMiami80 GPT Connect umożliwia generowanie unikalnych komentarzy przy użyciu cheatGPT, połączonego z OpenAI API. Użytkownik może skonfigurować różne opcje w panelu administratora, takie jak ustawienia AIengine, AutoGPT, i inne.

== Instalacja ==

1. Przejdź do katalogu `wp-content/plugins/` w swoim katalogu WordPress.
2. Wrzuć całą wtyczkę (katalog `koment-webmiami80-gpt-connect/`) do katalogu `plugins`.
3. Aktywuj wtyczkę poprzez panel administracyjny WordPress.

== Ustawienia ==

Po aktywacji wtyczki, przejdź do zakładki `Ustawienia` w panelu administracyjnym WordPress, aby skonfigurować opcje AIengine, AutoGPT, Kolejki, CRON i inne.

== Generowanie Komentarzy ==

W panelu administracyjnym znajdziesz sekcję do generowania komentarzy, gdzie możesz dostosować parametry generowania, ograniczenia, oraz śledzić statystyki i historię.

== Dodatkowe Opcje ==

1. **Duplikaty:** System sprawdzający, czy taki komentarz już istnieje, aby uniknąć powtórzeń.
2. **Statystyki:** Statystyki na temat ilości wygenerowanych komentarzy.
3. **Powiadomienia:** Powiadomienia dla administratora o zakończonych zadaniach generowania.
4. **Historia Generowania:** Historia wszystkich wygenerowanych komentarzy w panelu administracyjnym.
5. **Logi:** Logi dla debugowania i monitorowania działań wtyczki.

== Dezinstalacja ==

1. Deaktywuj wtyczkę poprzez panel administracyjny WordPress.
2. Przejdź do zakładki `Wtyczki` i użyj opcji `Usuń`, aby usunąć wtyczkę.
